Describe Your Acceptance Criteria

As a centre agent I need to be able to capture relationships between end users so that I can build rapport with our customers

Given I have viewed a person

When I click 'New' in the relationship section

Then I should see a data entry screen with the following fields (* indicates mandatory) Related End User* (Lookup), Relationship Type (Mother, Daughter)* (picklist), Start Date (date)

Given a relationship type of <blank> or a related end user of <blank>

When I try to save a relationship

Then I should see the error 'Please enter a value for the field' below the data entry point

Given an end date in the past

When I try to save a relationship

Then I should see the error 'You have entered a date in the past. Please try again' below the data entry point

Given a relationship with the same user, related user type

When I try to save a relationship

Then I should see the error 'There is already a <type> connection between these 2 people. Please try again' at the top of the page
